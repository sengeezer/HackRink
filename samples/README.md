# Samples

This folder includes one common-use example for each installed package and one combined example.

## Run setup

1. Use Node v20.15.1 (`nvm use` in this project).
2. Install dependencies: `npm install`.

## Run each sample

- Axios: `npm run sample:axios`
- Babel CLI + babel-preset-latest: `npm run sample:babel`
- bignumber.js: `npm run sample:bignumber`
- CoffeeScript: `npm run sample:coffee`
- node-fetch: `npm run sample:fetch`
- jQuery (browser): open `samples/jquery-example.html` in a browser
- lodash: `npm run sample:lodash`
- request: `npm run sample:request`
- TypeScript: `npm run sample:ts`
- underscore: `npm run sample:underscore`
- Combination sample (axios + node-fetch + lodash + underscore + bignumber.js + request): `npm run sample:combo`
